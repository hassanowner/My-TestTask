# Project File Structure

This project is organized to provide a **clear and structured overview** of all learning materials, exercises, and summaries related to JavaScript.  
The main folders are designed as follows:

- `course`: Contains the main lessons and corresponding challenges for each topic.  
- `Essential-extras`: Provides deeper insights, advanced examples, and additional topics not covered in the main course.  
- `Summaries`: Offers condensed versions of each topic for quick review and reference.  

> The following tree shows the **hierarchy of folders and files**, so you can quickly locate lessons, exercises, or summaries for any JavaScript topic.
